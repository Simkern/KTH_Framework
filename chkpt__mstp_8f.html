Detailed Description

Set of multi-file checkpoint routines for DNS, MHD and perturbation simulations.

This module writes different file sets for DNS (rsX...), MHD (rbX...) and perturbation (rpX...). In perturbation mode only single perturbation (NPERT=1) is supported right now.
